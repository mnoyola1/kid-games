# Rhythm Academy - Image Generation Prompts for Gemini

## Game Overview
Music rhythm game similar to Guitar Hero where notes fall down lanes and players must hit them at the right time. Some notes have educational questions for bonus points. Features 3 difficulty levels and 3 songs with different BPMs.

## Tips for Rhythm Academy Visual Style:

1. **High Contrast**: Notes must be instantly readable against background
2. **Timing Clarity**: Hit zone must be obvious and well-defined
3. **Beat Synchronization**: Visual effects should pulse with music tempo
4. **Color Coding**: Consistent lane colors throughout game
5. **Performance Feedback**: Immediate visual response to every action
6. **Music Theme**: Every element reinforces musical education theme

## Color Coding System:

### Note Lane Colors (Standard):
- **Lane 1**: Red (#F44336)
- **Lane 2**: Blue (#2196F3)
- **Lane 3**: Green (#4CAF50)
- **Lane 4**: Yellow (#FFEB3B)
- **Lane 5 (Medium)**: Orange (#FF9800)
- **Lane 6 (Hard)**: Purple (#9C27B0)

### Feedback Colors:
- **Perfect**: Gold (#FFD700)
- **Good**: Green (#4CAF50)
- **Miss**: Red (#F44336)
- **Question Note**: Purple (#9C27B0)

## Animation Frame Rates:

- Note fall speed: Varies by BPM (120-160)
- Hit feedback: 6-8 frames
- Particle effects: 4-6 frames loop
- Character idle: 2-3 frames loop
- Combo celebration: 8-10 frames

Generate each animation frame separately with consistent character and effect design.

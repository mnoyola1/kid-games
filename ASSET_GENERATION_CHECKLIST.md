## Pro Tips

### Batch Similar Assets
- Generate all enemies for a game in one session
- Keep style consistent by generating related items together
- Save prompts that work well for reuse

### Test Early
- Don't wait to generate ALL assets before testing
- Get characters in-game first
- Test gameplay feel before polishing

### Iterate Based on Gameplay
- Some assets look great but don't work in-game
- Kids' feedback is your best guide
- Polish visible elements first

### Track Progress
- Check off items as you generate them
- Mark items that need revision
- Note which prompts produced best results

**Notes:**

## GENERATION WORKFLOW

### Step 1: Prepare Workspace
```bash
# Create music folders for each game
cd "C:\Users\mnoyo\OneDrive\Documents\Personal\AI\games\kid-games\assets\audio"

mkdir canada-adventure\music
mkdir lumina-racer\music
mkdir word-forge\music
mkdir math-quest\music
mkdir pixel-quest\music
mkdir rhythm-academy\music
mkdir shadows-in-the-halls\music
```

### Step 2: Generate with Lyra 2
1. Go to: https://aitestkitchen.withgoogle.com/tools/music-fx
2. Copy/paste prompt from this document
3. Click "Generate"
4. Listen and evaluate quality
5. If satisfied, download MP3
6. Rename file appropriately (e.g., `main_menu_theme.mp3`)
7. Save to correct game music folder

### Step 3: Integrate into Game
1. Update `game-audio.js` or `game-config.js` with music file paths
2. Add music playback logic (if not already present)
3. Test in-game to ensure proper looping and transitions

### Step 4: Quality Check
- [ ] Music loops smoothly (no jarring cuts)
- [ ] Volume is appropriate (not too loud/quiet)
- [ ] Music doesn't distract from educational content
- [ ] Appropriate mood for game context
- [ ] Kid-friendly (ages 8-10)

## FILE NAMING CONVENTIONS

Use clear, consistent naming:

```
[context]_theme.mp3       - Main gameplay tracks
[area]_theme.mp3          - Specific area/level music
[event]_fanfare.mp3       - Short celebration tracks
[event]_music.mp3         - Event-specific tracks
game_over.mp3             - Game over screen
main_menu_theme.mp3       - Title screen

Examples:
- main_menu_theme.mp3
- atlantic_region_theme.mp3
- boss_battle_theme.mp3
- victory_fanfare.mp3
- game_over.mp3
```

## NOTES & TIPS

### For Lyra 2:
- Be specific about "instrumental" and "kid-friendly"
- Mention "loopable" for gameplay tracks
- Specify atmosphere clearly (welcoming, suspenseful, epic, etc.)
- Request specific instruments if needed (accordion, chiptune, etc.)
- Generation may take 30-60 seconds per track

### For Suno:
- Can add lyrics if desired (but we want instrumental)
- Better at specific genre requests
- Can regenerate with variations
- Download in highest quality available

### General Tips:
- Generate multiple versions, pick best
- Test loops before committing (import into Audacity to check seams)
- Keep originals backed up
- Consider master volume normalization across all tracks
- Get feedback from Emma & Liam on music preference

## Tips for Math Quest Visual Style:

1. **Educational Theme**: Incorporate mathematical symbols naturally into fantasy elements
2. **Color Coding**: Use consistent colors for different operations/difficulty levels
3. **Encouraging Atmosphere**: Even "enemies" should feel like learning challenges, not threats
4. **Progressive Difficulty**: Visual complexity should increase with game difficulty
5. **Clear Feedback**: Success and error states must be immediately recognizable
6. **RPG Conventions**: Follow classic RPG visual language for familiarity

## Combat Animation Sequence:

### Player Turn:
1. Player attack stance
2. Math spell casting
3. Problem display
4. Answer input
5. Correct: spell hits enemy
6. Wrong: player takes damage

### Enemy Turn:
1. Enemy attack wind-up
2. Problem display
3. Answer input
4. Correct: block enemy attack
5. Wrong: take damage

Generate each frame separately with consistent character design throughout battle.
